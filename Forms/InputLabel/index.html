InputLabel

InputLabel inserts a label on top of a form element component.

hidden property is used to toggle visibility with animations. For instance you may want to hide the label of an input when there is a placeholder and show the label when the input has some value.

Props

text
string | undefined
hidden
boolean | undefined

Use Cases

Default
<InputLabel text="Some input" />
Hidden
<InputLabel text="Some input" hidden />

Playground

<InputLabel text="Some input" hidden={false} />
