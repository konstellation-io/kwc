# ContextMenu

ContextMenu component allows the user to open a contextual menu on right (optionally left) click.

This menu shows actions that can be performed over the clicked element.

## Props and interfaces
#### Interfaces
##### MenuCallToAction
<Props of={MenuCallToActionIface} />
