# InfoMessage

InfoMessage component shows a text indicating some useful information. Typically used when some 
information is expected by for some reason (not an error) there is not any (for instance there is not 
enough available data).

## Use Cases

<Playground>
  <InfoMessage message="There is no information for the selected filters" />
</Playground>
