# Spinners

Spinner components allow the user to know that something is happening on the background and cannot be shown
while is not ready.

## Props
<Props of={SpinnerCircular} />

## Usage

This component typically block the render of the content that is not ready, to do that this components has 
to be returned earlier.

``` tsx
function Component() {
  const { data, loading } = useQuery(QUERY);

  // Interrupt the render process by returning this component
  if (loading) return <SpinnerCircular />;

  ...
}
```
